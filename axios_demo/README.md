# axios-learn

install Vant 2 for Vue 2 project
```
yarn add vant@latest-v2
```

引入vant组件
```
yarn add -D babel-plugin-import

// 对于使用 babel7 的用户，可以在 babel.config.js 中配置
module.exports = {
  plugins: [
    ['import', {
      libraryName: 'vant',
      libraryDirectory: 'es',
      style: true
    }, 'vant']
  ]
}
```

yarn install
